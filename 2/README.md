# 3D 직선형 스케치 툴 (3D Node-Link Simulator)

본 프로젝트는 상용 3D 물리 시뮬레이션 및 CAD 프로그램의 설계 시스템에 착안하여 구현하였습니다.
특히 게임 폴리브릿지에서 영감을 받아 pygame 라이브러리를 사용하여 제작하게 되었습니다.

1. 사용 라이브러리 및 버전 정보
* 선택한 외부 라이브러리: 'pygame-ce' (Community Edition | 일반적인 pygame이 python 버전과 맞지 않아 ce를 사용함.)
* 설치된 버전 'pygame-ce==2.5.3'

2. 파일 구조
```
2
    main.py
    requirements.txt
    README.md # (본 파일)
```

3. 설치 및 실행 방법
```
pip install -r requirements.txt
```
위 방식으로 라이브러리를 설치한 후 main.py를 실행합니다.

4. 조작 가이드 (Controls)
- 카메라 회전: 키보드 W, A, S, D 또는 방향키(누르고 있으면 연속 회전)
- 카메라 줌: 마우스 휠(스크롤)
- 밀도(배율) 조절: 좌측 상단의 버튼 클릭
- 링크(선) 생성 3D 공간상의 점(Node)을 좌클릭 한 후, 다른 점을 좌클릭하여 연결
- 링크(선) 삭제: 삭제할 선 위에 마우스를 올린 뒤(빨간색 하이라이트가 칠해져있다.) 우클릭 또는 Delete 키 입력
- 그리기 취소: 점을 하나만 찍은 상태에서 Backspace 또는 우클릭
